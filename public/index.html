Get cWBTC for Fiber CCH testing

cWBTC is a test-only CKB UDT used to model wrapped BTC accounting. Use it to test Fiber Cross-Chain Hub payments from CKB UDT channels into Lightning testnet invoices.

Testnet only. cWBTC has no mainnet value and is issued only for this demo environment.

Why this token exists

A test asset for the CCH payment path

Fiber CCH treats this UDT as wrapped BTC by convention: one raw cWBTC unit maps to one satoshi in the demo. That lets developers exercise the route without touching mainnet BTC.

Use it with Fiber CCH

Connect to Bottle and open a cWBTC channel

Bottle is the public Fiber testnet node used by the CCH demo. It is configured to accept cWBTC channels, so developers can connect by pubkey and open a UDT channel without operating a public relay first.

fiber-testnet-public-bottle

Auto-accepts cWBTC UDT channels for testnet demos.
01

Whitelist cWBTC

ckb:
  udt_whitelist:
    - name: "cWBTC"
      script:
        code_hash: "0x25c29dc317811a6f6f3985a7a9ebc4838bd388d19d0feeecf0bcd60f6c0975bb"
        hash_type: type
        args: "^0x9a1086531ed6dc69e0bd44cef5278e03faf3015b31aff60b08fb87663ce8507100000000$"
      auto_accept_amount: 1000000000
02

Enable CCH wrapped BTC

cch:
  wrapped_btc_type_script_args: "0x9a1086531ed6dc69e0bd44cef5278e03faf3015b31aff60b08fb87663ce8507100000000"
03

Connect to Bottle

fnn-cli peer connect_peer \
  --pubkey 02b6d4e3ab86a2ca2fad6fae0ecb2e1e559e0b911939872a90abdda6d20302be71

Fiber v0.8.0 and newer can connect by pubkey after graph data resolves the public node address.
